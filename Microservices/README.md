# Microservices DeepSkilling Exercises

This repository contains the Microservices exercises and hands-on implementations completed as part of the Cognizant Digital Nurture 4.0 DeepSkilling Program.

## Modules

### 0. Sample Microservices Exercises
Implemented basic microservices using Spring Boot and REST APIs.

### 0. Sample Microservices Load Balancing Exercises
Implemented microservice communication and load balancing concepts using Spring Boot and Spring Cloud.

### 1. Microservices using Spring Boot 3 Exercises
Implemented Spring Boot 3 based microservices including:

- User and Order Management
- Inventory Service Discovery using Eureka Server
- API Gateway with Customer and Billing Services
- Resilient Payment Microservice using Resilience4j Circuit Breaker and Fallback

### 2. Microservices with API Gateway
Implemented API Gateway concepts for routing requests between different microservices.

### 3. Microservices Composite Hands-On
Implemented composite microservice concepts and service communication.
