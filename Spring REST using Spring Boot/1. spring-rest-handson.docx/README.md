# Spring REST using Spring Boot

This repository contains the Hands-On exercises completed for the **Spring REST using Spring Boot** module as part of the **Cognizant Digital Nurture Program**.

## Hands-On Exercises Completed

- ✅ HandsOn 1 - Spring Boot Application Setup
- ✅ HandsOn 2 - Load SimpleDateFormat from Spring XML Configuration
- ✅ HandsOn 3 - Logging using SLF4J
- ✅ HandsOn 4 - Spring Bean Configuration using XML
- ✅ HandsOn 5 - Bean Scopes (Singleton & Prototype)
- ✅ HandsOn 6 - Collection Injection (List of Countries)

## Technologies Used

- Java 21
- Spring Boot 3.5
- Spring Core
- Maven
- IntelliJ IDEA
