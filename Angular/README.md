## Project Overview

This project was developed as part of the **Cognizant Digital Nurture 5.0 Deep Skilling Program**.

The application is a **Student Course Portal** built using **Angular 18 (Standalone Components)**. Each Hands-On exercise extends the same project by gradually adding new Angular concepts and features.

# How to Run

### Install Dependencies

```bash
npm install
```

### Start JSON Server

```bash
npx json-server --watch db.json --port 3000
```

### Run Angular Application

```bash
npx ng serve
```

Open:

```
http://localhost:4200
```

---

# Run Unit Tests

```bash
npx ng test
```

---

# Generate Code Coverage

```bash
npx ng test --code-coverage
```
